# Let's Speak AI - تطبيق تعلم اللغة الإنجليزية

تطبيق تفاعلي لتعلم اللغة الإنجليزية باستخدام الذكاء الاصطناعي، مصمم لثلاثة أنواع من المستخدمين: الطلاب، المدربين، والمديرين.

## 🎨 الثيم والألوان

تم تصميم التطبيق باستخدام نظام ألوان مخصص:

- **أخضر طوخ**: `#003F3E` - اللون الأساسي الداكن
- **أخضر تركواز**: `#02BAAA` - اللون الأساسي
- **برتقالي**: `#FF914D` - اللون الثانوي
- **رصاصي**: `#96A8AD` - اللون المحايد
- **رمادي فاتح**: `#D8D0C5` - لون الحدود
- **أبيض مائل للرمادي**: `#E6E5E1` - لون الخلفية الفاتح

## 👥 أنواع المستخدمين

### 1. الطالب (USER)

- **الوظائف الرئيسية**:
  - تعلم كلمات جديدة
  - إكمال المسارات التعليمية
  - التحدث مع الذكاء الاصطناعي
  - توليد القصص والاختبارات
  - تتبع التقدم والإنجازات

### 2. المدرب (TRAINER)

- **الوظائف الرئيسية**:
  - عرض قائمة جميع الطلاب
  - مراقبة تقدم الطلاب
  - عرض الكلمات الصعبة لكل طالب
  - إحصائيات مفصلة عن أداء الطلاب
  - نظرة عامة على جميع الطلاب

### 3. المدير (ADMIN)

- **الوظائف الرئيسية**:
  - إدارة جميع المستخدمين
  - تغيير أدوار المستخدمين
  - إحصائيات شاملة للنظام
  - مراقبة أداء المدربين والطلاب
  - إدارة المحتوى التعليمي

## 🛠️ التقنيات المستخدمة

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS + Custom Theme
- **Icons**: Heroicons
- **State Management**: React Context
- **HTTP Client**: Axios
- **Routing**: React Router DOM
- **Notifications**: React Hot Toast
- **Animations**: Framer Motion

## 🎯 خدمات API

### المصادقة (Authentication)

- `POST /api/auth/register` - تسجيل مستخدم جديد
- `POST /api/auth/login` - تسجيل الدخول
- `GET /api/auth/profile` - جلب الملف الشخصي

### الكلمات (Words)

- `GET /api/words` - كلمات المستخدم
- `POST /api/words` - إضافة كلمة جديدة
- `POST /api/words/:id/mark-known` - تعليم كلمة كمحفوظة
- `GET /api/words/stats` - إحصائيات الكلمات

### المدرب (Trainer)

- `GET /api/trainer/students` - قائمة الطلاب
- `GET /api/trainer/overview` - نظرة عامة
- `GET /api/trainer/students/:id` - تفاصيل طالب

### المدير (Admin)

- `GET /api/admin/stats` - إحصائيات النظام
- `GET /api/admin/users` - جميع المستخدمين
- `PUT /api/admin/users/:id/role` - تغيير دور المستخدم
